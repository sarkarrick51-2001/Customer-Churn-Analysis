# 📊 Customer Churn Analysis

## 📌 Objective

Analyze customer data to identify patterns and factors leading to customer churn, and provide actionable insights to improve retention.

---

## 🛠️ Tools Used

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* SQL
* Excel

---

## 📂 Dataset

* Customer data including demographics, subscription details, and usage behavior
* Contains features like tenure, monthly charges, contract type, and churn status

---

## 🔍 Analysis Performed

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Churn Rate Analysis
* Feature Impact Analysis (tenure, contract, charges)
* Customer Segmentation

---

## 📈 Key Insights

* Customers with **short tenure** have higher churn rates
* **Monthly contract users** churn more than yearly subscribers
* Higher **monthly charges** are linked to increased churn
* Customers without **long-term commitment** are at risk

---

## 💡 Business Impact

* Encourage long-term subscriptions with discounts
* Target high-risk customers with retention offers
* Improve onboarding experience for new users
* Optimize pricing strategies
